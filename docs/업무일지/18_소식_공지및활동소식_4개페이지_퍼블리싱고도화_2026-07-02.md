## 1. 금일 작업 내역

### 1) 공지사항 목록 (`notice.html`) 및 상세 (`notice/detail.html`) 정밀 퍼블리싱
- **피그마 실측 사양 동기화**:
  - KV 영역: 기존의 CSS 배경 지정을 걷어내고, HTML 마크업 상에 실사이즈 가로/세로 속성을 부여한 `<img>` 태그를 직접 삽입하였습니다. CSS 스타일로 `.news-kv .kv-img`에 `position: relative;` 및 `left: 50%; top: 5.0rem; transform: translateX(-50%);`, `max-width: none;`을 적용하여 상단에서 50px 아래에 실사이즈가 왜곡 없이 정밀하게 얹히도록 하고, 타이틀과 디스크립션 텍스트 랩을 `position: absolute;`로 띄워 정중앙 정렬 처리하였습니다.
  - 검색바: 배경색을 제거하여 **투명하게 정합**(`background-color: transparent;`)시키고, 단일 텍스트 인풋 필드 형태로 맞추었습니다.
  - 목록: 피그마 시안에 없는 **캘린더 SVG 아이콘을 배제**하였습니다. 또한, 임의 제작했던 New 뱃지를 걷어내고 **`src/images/common/icon_badge_new.png` 에셋으로 교체하여 날짜 좌측에 위치**시켰습니다. 추가로 **날짜 우측에 `icon_arrow_right.png` (22px * 16px) 에셋을 연동**하였습니다.
  - 상세페이지: 상세 헤더, 본문 포스터 이미지(`img_notice_view.png`, W: 800px, H: 500px 명시로 CLS 차단)와 본문 텍스트, 이전/다음 글 화살표(22px * 16px) 및 목록 버튼 배치.
