## 1. 금일 작업 내역

### 1) 후원 상세 정보 요약 표 2개로 이중 분할
- **기존 상태**: 하나의 큰 단일 요약 표(`detail-info-table`) 내에 구분, 기부일, 금액, 결제방법이 모두 섞여 있었음.
- **수정 상태**: 피그마 2개 프레임 배치에 맞춰 **2개의 개별 테이블**로 마크업을 분할 구성:
  - **테이블 1**: 구분, 기부 시작일, 기부 종료일, 해지 신청 관련 정보
  - **테이블 2**: 후원금액, 결제방법 / 결제일, 금액 변경, 결제 변경 관련 정보
- **테이블 간 간격**: 두 테이블이 인접할 경우 `.detail-info-table + .detail-info-table` 에 `margin-top: 2rem (20px)` 을 추가 정의하여 간격을 주었습니다.

### 2) 테이블 셀 높이(Height: 86px) 및 패딩 정밀 동기화
- **피그마 실측 사양**: th/td 셀의 Bounding Box Height `86px`, padding `Top=30px, Bottom=30px, Left=24px, Right=24px`.
- **수정 사양 (`login.css`)**: 
  - `.detail-info-table th` 및 `td` 에 **`padding: 3rem 2.4rem;`** 및 **`height: 8.6rem;`**, **`box-sizing: border-box;`**를 적용하여 셀 내부 텍스트와 버튼이 피그마 시안과 한 치의 오차도 없는 86px 높이 내에 정교하게 정렬되도록 하였습니다.

### 3) 페이지네이션 간격 및 비활성 배경 투명화
- **피그마 실측 사양**: 
  - 3개 페이지 그룹 간 여백(itemSpacing)은 **`50px` (5.0rem)**.
  - 숫자 그룹 내부의 버튼 간 여백은 **`12px` (1.2rem)**.
  - 비활성 버튼은 채우기가 없고 오직 보더만 있는 상태 (Fills 없음).
- **수정 사양 (`login.css`)**:
  - `.pagination` 의 `gap` 을 `2rem` ➡️ **`5rem`** 으로 대폭 넓혔습니다.
  - `.page-group-arrow, .page-group-numbers` 의 `gap` 을 `0.8rem` ➡️ **`1.2rem`** 으로 조정하였습니다.
  - `.page-btn` 의 기본 `background-color` 를 `#ffffff` ➡️ **`transparent`** 로 수정하여 연노랑 본문 배경에서 비활성 번호 버튼의 하얀 사각형 상자가 튀어 보이지 않고 투과되도록 정밀 조율했습니다.
  - `.page-btn.active` 에는 기존처럼 빨간색 활성 배경이 유지되도록 확실하게 오버라이딩 하였습니다.
