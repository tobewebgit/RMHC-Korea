## 3. 개발팀 인수인계 고려사항
* **마이페이지 LNB 스크립트**: LNB 인클루드 내에 내장된 스크립트가 브라우저 URL을 매칭하여 현재 활성화된 메뉴 항목에 동적으로 클래스를 바인딩하므로, 라우팅 설계 시 URL과 data-menu 속성 맵핑을 참고해 주십시오.
* **모달 딤드 스크롤 락**: 모달 활성화 시 `body`에 `menu-open` 클래스를 추가/제거하여 모바일에서의 배경 스크롤 현상을 방지하고 있습니다.
* **우편번호 검색**: `apply.html` 하단에 가상의 `sampleAddressSearch` 우편번호 팝업 모의 로직이 연동되어 있어, 추후 실 서비스 적용 시 카카오 우편번호 API 등의 실 팝업 모듈로 교체하여 사용할 수 있습니다.
