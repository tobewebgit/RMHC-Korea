## 🛠️ 금일 작업 내역

### 1. 공통 탭 활성화 스크립트 모듈 구축
- **신규 파일**: `src/js/components/tab.js`
  - `data-tab-target` 속성을 기반으로 동작하는 범용 탭 스크립트 구현.
  - 중첩 탭(Nested Tabs) 간 충돌 방지를 위해 클릭 대상의 탭 콘텐츠 클래스명(예: `.tab-content`, `.sub-tab-content`, `.payment-tab-content`)을 확인하여 정확히 일치하는 자매 노드들만 display 제어하도록 예외 처리 완비.
  - 탭 버튼 그룹의 부모 컨테이너가 다른 마크업 구조를 가질 경우를 대비해 `data-tab-group` 및 유연한 셀렉터 체이닝 적용.
- **연동**: `src/js/common.js`에 `initTabs` 임포트 및 초기 실행 정의 완료.

### 2. 결제수단 변경 팝업 데이터 속성 교정
- **수정 파일**: `includes/popups/mp_p_03.html`
  - 5종 결제수단 탭 버튼의 속성을 기존 `data-tab="..."`에서 공통 규격인 `data-tab-target="..."`으로 리네이밍하여 공통 탭 스크립트가 온전히 적용되도록 수정.

### 3. 마이페이지 탭 마크업 보정 및 인라인 스크립트 제거
- **수정 파일**: `mypage/mypage.html`
  - 내부 서브 탭(전체/정기/일시) 버튼의 속성을 기존 `data-sub-tab`에서 `data-tab-target` 공통 명세로 전환.
  - 720라인 이하의 인라인 스크립트 중 탭 버튼 토글, 서브 탭 토글, 결제수단 탭 토글과 관련된 약 140라인 분량의 중복 인라인 코드 완전 제거.
  - 눈 아이콘 비밀번호 보기 토글 및 공통 모달 팝업 제어 전용 자바스크립트만 분리 독립 구동 확인.

### 4. 자주 묻는 질문(FAQ) 마크업 및 스타일 피그마 팩트 동기화
- **수정 파일**: `faq/faq.html`
  - 피그마 시안에 존재하지 않는 **Q 아이콘, New 뱃지, 날짜, 화살표 아이콘을 전면 배제**하고 실제 디자인 명세 그대로 리팩토링.
  - 카테고리 탭("로날드 맥도날드 하우스 양산", "기부금영수증", "후원관련") 마크업 및 공통 탭 구조 연동.
  - 질문 영역(NanumSquare Neo 28px, 800)의 우측에 아코디언 상태를 표현하는 플러스/마이너스 토글 버튼(`.faq-toggle-btn`) 배치.
  - 답변 영역(Pretendard 22px, 400) 마크업 및 아코디언 단일 개폐 스크립트 바인딩.
- **수정 파일**: `src/css/faq.css`
  - 피그마 Node ID `2405:105319` 분석 사양 100% 매치 스타일 시트 작성.
  - 최대 너비 120rem(1200px) 중앙 정렬 및 여백 구현.
  - **탭 영역 전체 회색 가로선 제거** (피그마에 없는 요소 배제).
  - **탭 버튼 텍스트 색상 통일**: 비활성 상태 탭 또한 피그마 실측과 동일하게 **절대 블랙(`#000000`)으로 통일**하여 임의의 회색 대비 배제.
  - **아코디언 보더 및 배경 리팩토링**:
    - 개별 항목의 **하단 보더(`border-bottom`)를 완전히 제거**하고, 상단 보더(`border-top`)만 남기어 첫 항목을 제외한 연속 요소들의 상단 구분선만 표시하도록 교정.
    - 활성 아코디언 답변 영역의 **배경색(기존 라벤더그레이)과 질문 영역과의 경계 점선을 완전히 제거**하여 피그마 실측 그대로 투명 텍스트 구조로 변경.
  - **플러스/마이너스 CSS 드로잉 구현**: 가상 요소 `::before`(가로선) 및 `::after`(세로선)를 배치하고, 질문 활성화 시(`.faq-q.active`) 세로선을 90도 회전 및 페이드아웃하여 깔끔하고 세련된 플러스/마이너스 모션 재현.
  - 모바일 반응형 뷰포트(max-width: 1024px) 폰트 크기 및 패딩 보정 스타일 구현 완료.
