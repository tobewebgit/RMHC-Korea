### 2) 자원봉사 신청 페이지 신규 구축 및 개별 스타일 구축
* **자원봉사 신청 화면 신규 퍼블리싱 (`volunteer.html`)**
  - **개인 vs 기업/단체 신청 탭 전환**: 웹 접근성 표준 규격(`role="tablist"`, `aria-selected` 등)을 준수한 탭 전환 메뉴 구성 및 바닐라 JS 바인딩.
  - **직접 마크업 캘린더**: 라이브러리 의존성을 제거하고 시안과 완전 동일하게 커스텀하기 위해 `table` 구조의 캘린더를 정밀 구현 (오늘 날짜, 선택일, 불가일 스타일링).
  - **VMS 연계 안내**: 하단 영역에 봉사 실적 등록 안내 문구를 포함한 시인성 높은 알림 카드(`vms-notice-card`) 탑재.
* **자원봉사 도메인 전용 CSS 구축 (`src/css/volunteer.css`)**
  - 봉사활동을 상징하는 그린/민트 테마 컬러(`--accent: #10b981`)를 적용하여 싱그럽고 정돈된 카드 뷰와 달력 스타일 설계.
  - 모바일 뷰포트(768px 이하)에서 캘린더와 우측 일정 요약 박스가 세로 1열로 매끄럽게 재배치되도록 반응형 미디어 쿼리 구축.

## 3. 개발팀 인수인계 고려사항
* **정기/일시 탭 전환**: 상단의 `.donate-tabs`는 `href`를 통한 완전한 페이지 이동 방식으로, 각 정기/일시/물품 후원이 별도 HTML로 독립 배치되어 있어 유지보수가 용이합니다.
* **기부금 영수증 아코디언**: `receiptChips` 클릭에 따라 `.accordion-content`에 `.active` 클래스를 추가/제거하여 슬라이딩을 제어합니다.
* **자원봉사 캘린더**: 선택된 날짜의 `day-btn` 클릭 시 hidden input(`#input-date`)으로 `YYYY-MM-DD` 데이터가 폼에 동적으로 매핑됩니다.
