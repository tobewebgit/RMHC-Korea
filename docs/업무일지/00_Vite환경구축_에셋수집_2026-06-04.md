## 2. 특이사항 및 이슈 사항 (Issues & Mitigations)

*   **대형 프레임 렌더링 서버 지연 이슈 우회**
    *   **현상**: 피그마 API를 통해 캔버스 내의 전체 고해상도 PNG 시안을 번들로 다운로드 요청할 때, 피그마 서버의 렌더링 렉 및 지연으로 인해 API 커넥션이 홀딩 및 타임아웃을 유발하는 문제 발생.
    *   **조치**: `AbortController` 기반의 12초 타임아웃 세션 차단 설정을 다운로더 스크립트에 탑재. 무한 대기를 발생시키는 특정 프레임(3개 파일)은 안전하게 건너뛰고 다음 파일로 스킵하도록 설계하여, 정체 없이 9개의 핵심 레이아웃 시안 및 639개의 UI 자산을 안전하게 수집 완료.
    *   **안전 조치**: 개발 분석에는 피그마 API 토큰과 파일 링크가 연계되어 있으므로, 굳이 다운로드되지 않은 무거운 이미지가 없어도 노드의 폰트 사이즈, CSS 수치, 레이아웃 좌표값을 JSON 데이터로 정밀하게 직접 해석할 수 있어 마크업 완성도에는 영향 없음.
